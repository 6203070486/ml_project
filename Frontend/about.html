About the Project

Built for Smarter Farming

AquaAI is a student project that combines Machine Learning with a modern web interface to predict whether a crop field needs irrigation — helping farmers conserve water and improve yield efficiency.

How It Works

The system takes four environmental inputs — Temperature, Humidity, Soil Moisture, and Rainfall — and passes them to a trained Random Forest classifier via a Flask REST API.

The model returns a binary prediction: Irrigation Needed or No Irrigation Needed, along with a confidence score visualized as a doughnut chart. This helps farmers make data-driven decisions instead of relying on guesswork.
